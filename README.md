# ぽんず Daily Checklist

毎日のやることをシンプルに管理できる PWA（プログレッシブウェブアプリ）です。

## 機能

- チェックリスト形式でタスクを管理
- ホーム画面に追加してアプリのように使える（PWA 対応）
- スマートフォン・PC どちらでも動作

## 使い方

[https://kibiyakku17.github.io/ponzu-todo-app/](https://kibiyakku17.github.io/ponzu-todo-app/) をブラウザで開くだけで使えます。

ホーム画面への追加:

- **iOS（Safari）**: 共有ボタン →「ホーム画面に追加」
- **Android（Chrome）**: メニュー →「ホーム画面に追加」

## 技術スタック

- HTML / CSS / JavaScript（フレームワークなし）
- Service Worker による PWA 対応
