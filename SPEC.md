# ぽんずTodo 仕様書（現状まとめ / v3.2時点）

- リポジトリ：`ponzu-todo-app`
- 公開URL：https://kibiyakku17.github.io/ponzu-todo-app/
- 構成：単一 `index.html` のPWA（ビルド不要）。データはすべて `localStorage`（この端末内のみ）。
- デザイン：ベージュ背景＋セージ/コーラル/スカイ/ラベンダー/アンバー/ローズのアクセント、Zen Maru Gothic（見出し）＋Zen Kaku Gothic New（本文）。
- コンセプト：**考えるのはAI。やるのはあなた。** AIで整理したリストを、コピペだけで登録できるTodoアプリ。

> バージョン履歴（要点）
> - **v3**：ホーム/管理の2画面分離、「今日だけ」タスク、種別/時間帯/カテゴリタグの一括投入
> - **v3.1**：AIコピペTodoのUI改善（ボタン名「AIリストを追加」、タグライン、ウェルカムカード、初回チュートリアル）
> - **v3.2**：ごみ出し設定のAIコピペ化・初期値を未設定に

## 1. 全体構造（2画面＋下部タブ）

画面下部に固定ナビ（`position:fixed`）があり、2つの画面を切り替える。

| タブ | 役割 |
|---|---|
| 🏠 **ホーム** | 今日やることを見る・チェックする画面（追加/編集はしない） |
| ⚙️ **管理** | Todo・カテゴリ・ごみ出しの追加/編集/削除を行う画面 |

「種別」（今日だけ/毎日/週/月）の切り替えは管理ページ内のタブで行う。ホームは種別を切り替えず、該当する全セクションを同時に縦表示する。

---

## 2. ホーム画面

- **ヘッダー**：アプリ名「ぽんずTodo」／今日の日付・曜日／タグライン「考えるのはAI。やるのはあなた。」／肉球メーター（進捗%）／一言メッセージ（進捗で変化）
- **ごみ出しバナー**：曜日に設定があれば自動表示。未設定の曜日は「今日のごみ出しはありません」表示。⚙️タップで管理ページの「ごみ出し」へ遷移
- **セクション**（データがある場合のみ、上から順に縦表示）：
  - 📌 **今日だけ**：対象日が今日 かつ 未完了 のタスク
  - 🔁 **毎日**：時間帯別（🌅朝／☀️昼休み／💻仕事中／🌇仕事終わり／🕐時間未設定）
  - 🎀 **今週のタスク**：土曜日のみ表示
  - 🌙 **今月のタスク**：月初（第1土曜）のみ表示
- ホームは**チェック専用**（個別の追加/編集ボタンは置かない）
- 各セクション見出しに **−/＋ボタン**があり、そのセクションの完了済みの表示/非表示を切り替えられる

### チェックの挙動
- **毎日/週/月**：チェック状態は日付キー（`daily_YYYY-MM-DD` 等）で保存。完了済みは取り消し可能。
- **今日だけ**：チェックすると `completed=true` になり**即座に非表示**。完了履歴は残さず、取り消し手段もなし（単発タスクのため）。

### ウェルカムカード（Todo0件/初回/全削除直後）
- 「🐾 AIと一緒にTodoを作ろう」＋3ステップ（AIで整理→AIリストを追加に貼り付け→自動カテゴリ分け）＋「📋 AIリストを追加を開く」ボタン
- Todoが1件でも登録されると非表示。0件に戻れば再表示
- ごみ出し未設定 かつ 未スキップ時は、ウェルカムカード下に「🗑️ ごみ出しも設定できます（お好みで）」の案内（「🤖 ごみ出しを設定する」「あとで設定する」）を併記。「あとで」を押すと以後非表示（`trash_hint_dismissed_v3`）

## 3. 「今日だけ」タスク

毎日/週/月のどれにも当てはまらない単発タスク（病院・美容院・役所手続き・宅配受取など）。

- 登録項目：内容／メモ（任意）／**対象日**（デフォルト今日）／カテゴリ
- ホームには「対象日が今日 かつ 未完了」のものだけ表示
- チェックで即非表示・完了扱い
- **期限切れ**（対象日が過去で未完了）：ホームからは消えるが**管理ページの「今日だけ」一覧には残る**。「対象日：6/15・期限切れ」とコーラル色で表示され、編集で対象日の再設定や削除ができる
- データ：`todo_items_today_v3` に各タスクが `targetDate` と `completed` フラグを持つ

## 4. 管理ページ

上部の横スクロールタブで以下を切り替える：

`📌今日だけ` / `🔁毎日` / `🎀週` / `🌙月` / `🏷️カテゴリ` / `🗑️ごみ出し`

### Todo種別（今日だけ/毎日/週/月）の各タブ
- 操作行：**＋個別追加** ／ **📋 AIリストを追加** ／ **チェックをリセット**（今日だけ以外）
- ボタン付近に常時表示の説明文「💡 ChatGPTやClaudeで作ったリストを貼り付けると、カテゴリや時間帯ごとにまとめて登録できます。」
- カテゴリ別のセクションでタスク一覧を表示。各タスクの ✏️ で編集、各カテゴリ見出しの ⚙️ でカテゴリ編集
- カテゴリが0件でも追加フォームは開ける（カテゴリは追加フォーム内やAIリスト追加で作れる）
- 空状態は「📋 AIで作ったリストを貼り付けて、まとめて登録できます。」＋「📋 AIリストを追加」ボタン

### 個別追加・編集モーダル
- 共通：内容／メモ／カテゴリ
- 種別による差し替え：
  - **今日だけ** → 対象日（日付入力）
  - **毎日** → 時間帯（朝/昼休み/仕事中/仕事終わり/時間未設定）
  - **週/月** → 時間帯・日付ともなし
- カテゴリ選択の末尾に「＋ 新しいカテゴリを作る」。選ぶとカテゴリ追加に進み、作成後に元のフォームへ戻り自動選択
- **モーダルは背景タップ・Escapeでは閉じない**（入力中の誤操作で消えるのを防ぐ）。閉じるのはキャンセル/保存のみ

## 5. AIリストを追加（一括テキスト貼り付け）

AIチャットで生成した箇条書きを、コピペで一括登録する機能（v3までの「まとめて追加」を改称）。

### モーダル構成
- 先頭に折りたたみ「🤖 AIにお願いする例（プロンプト）を見る」（タップで展開、コピーボタン付き）
- 貼り付けテキストエリア → 即時プレビュー

### パース仕様
- 1行＝1タスク。空行は無視
- 行頭の `-` `・` `*` `①` `1.` `- [ ]` `□` などの記号・番号・チェックボックスは自動除去
- **種別タグ**：`[今日だけ]`(`[今日]`) / `[毎日]` / `[週]`(`[週次]`) / `[月]`(`[月次]`)
  - 単独行で書くと、以降の行がその種別に振り分けられる
  - 種別タグが一度も無ければ、管理ページで選択中の種別に全件登録（後方互換）
  - `[今日だけ 6/20]` のように対象日を添えられる（省略時は貼り付け日）
- **時間帯タグ**（毎日タスク用）：`[朝]` / `[昼休み]`(`[昼]`) / `[仕事中]` / `[仕事終わり]`(`[夕方]`/`[夜]`) / `[時間未設定]`
  - 単独行で書くと、以降の毎日タスクにその時間帯が適用される
- **カテゴリタグ**：`[家事]` のように、種別/時間帯以外のブラケットはカテゴリ名として認識
  - 単独行 → 以降の行のカテゴリ／インライン（`[家事] 洗濯`）→ その行のみ
  - 既存カテゴリ名と部分一致で自動振り分け。存在しないカテゴリは「✨新しく作るカテゴリ」として絵文字を指定でき、追加時に自動生成

### プレビュー
- 各行に **種別／時間帯／カテゴリ** の3セレクトが付き、個別に修正可能
- 時間帯セレクトは種別が「毎日」のときだけ有効（他種別では無効化）
- 「＋作成」を選べばその場で新カテゴリも作れる

### 入力例
```
[今日だけ]
[家事]
病院の予約電話をする

[毎日]
[朝]
[健康]
ストレッチをする
[仕事中]
メールチェック

[週]
[ぽんず]
ブラッシング
```

### AIプロンプト（モーダル内にコピーボタン付き）
```
今日やること・今週やること・今月やることを整理してリストアップして。

条件：
・1行に1つのタスクだけ書く
・「-」から始める
・見出しは [今日だけ][毎日][週][月] を使う
・カテゴリは [家事] のように記載する
・毎日は [朝][昼休み][仕事中][仕事終わり] も付ける
```

## 6. カテゴリ

- 名前・絵文字・カラー（6色から選択）で作成。編集・削除も可能
- 削除時、紐づくTodoがあれば件数を確認ダイアログで通知（一緒に削除）
- 今日だけ/毎日/週/月の全種別で**共通のカテゴリ一覧**を使用
- **予約語はカテゴリ名にできない**：`今日だけ` `今日` `毎日` `週` `月` `週次` `月次` `朝` `昼` `昼休み` `仕事中` `仕事終わり` `夕方` `夜` `時間未設定`

---

## 7. 進捗・リセット

- **肉球メーター**（14個）は「毎日」タスクの完了率を基準に計算（今日だけ/週/月は含めない）
- **完了済み非表示**：ホームの各セクション単位で −/＋ トグル
- **リセット**：管理ページの毎日/週/月タブから、その種別のチェック状態のみ解除（タスク自体は消えない）

## 8. ごみ出し設定（v3.2でAIコピペ化）

- **初期状態は全曜日「未設定」**（地域固有の固定値は持たない）。未設定の曜日はホームにバナーを表示しない
- 設定方法は2つ（管理ページ「🗑️ごみ出し」タブ）：
  - **方法A：AIに聞いて貼り付け（推奨）**「🤖 AIに聞いて貼り付け」ボタン → モーダルでプロンプト例（コピー付き）を見て、AIの回答を貼り付け → 曜日ごとにパース＆プレビュー（色見本付き）→ 曜日別に手修正して保存
  - **方法B：手入力で編集**「⚙️ 手入力で編集」で曜日ごとに直接入力（フォールバック）
- **パース仕様**：行頭の `[月]`〜`[日]` 曜日タグを認識し、後ろのテキストをその曜日の収集物に。同じ曜日が複数回出たら後勝ちで上書き。曜日タグの無い曜日は未設定のまま
- **表示色の自動判定**（品目テキストのキーワード／プラを資源より先に判定）：
  - 燃やす・可燃 → coral／缶・びん・ペットボトル → sky／プラ → sage／古紙・段ボール・資源 → amber／その他 → lavender
- **既存データ保護**：`trash_schedule_v3` に保存済みの値があればそれを優先し、上書きしない（移行処理なし）
- AIプロンプト（ごみ出し用）：
```
私が住んでいる地域のごみ収集ルールを教えてください。
以下の条件で、曜日ごとに分けて書いてください。

条件：
・出力は曜日タグと品目だけにする（説明や前置きは不要）
・曜日タグは [月][火][水][木][金][土][日] を使う
・収集がない曜日は省略してよい
・複数の品目がある場合は「・」で区切る

例の書式：
[月] 燃やすごみ
[火] 缶・びん・ペットボトル
[水] 古紙・段ボール

私の地域：（ここに市区町村名や地域名を書く）
```

## 10. データ構造（localStorage）

| キー | 内容 |
|---|---|
| `todo_categories_v3` | カテゴリ一覧 `{id,name,emoji,color,soft}` |
| `todo_items_today_v3` | 今日だけタスク `{id,main,note?,targetDate,categoryId,completed}` |
| `todo_items_daily_v3` | 毎日タスク `{id,main,note?,time,categoryId}` |
| `todo_items_weekly_v3` | 週タスク `{id,main,note?,categoryId}` |
| `todo_items_monthly_v3` | 月タスク `{id,main,note?,categoryId}` |
| `trash_schedule_v3` | ごみ出し設定（曜日index→`{label,mark,cls}` または null） |
| `daily_YYYY-MM-DD` 他 | 毎日/週/月のチェック状態（週=`weekly_YYYY-Wxx`、月=`monthly_YYYY-MM`） |
| `todo_tutorial_seen_v3` | チュートリアル既読フラグ |
| `trash_hint_dismissed_v3` | ウェルカムのごみ出し案内スキップフラグ |

※ v2（`_v2`）との互換は持たず、v3では空からスタート。
